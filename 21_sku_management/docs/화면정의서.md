# 화면정의서

## 1. 관리자 공통

### 진입 경로

```text
/admin/index.html
```

### 메뉴 구조
- SKU 매핑
- 세트/번들

### 모듈 구조
화면은 모듈 단위로 추가한다.

```text
public/admin/modules/manifest.json
public/admin/modules/{module-name}.module.js
```

새 기능을 추가할 때는 기존 모듈 파일을 직접 수정하지 않고 새 모듈을 추가한 뒤 manifest에 등록한다.

## 2. SKU 매핑 화면

### 경로

```text
/admin/index.html#sku-mapping
```

### KPI
- 총 옵션수
- 매핑완료
- 미매핑
- 충돌

### 필터/액션 바
스크롤해도 화면 상단에 고정한다.

구성:
- 플랫폼 필터
- 상태 필터
- 검색어 입력
- 보기 방식
- 검색 버튼
- 선택 일괄 매핑 버튼
- 입력된 SKU 일괄저장 버튼

### 보기 방식

```text
옵션별
상품별 그룹
```

### 옵션별 테이블 컬럼
- 선택
- 플랫폼
- 플랫폼 상품번호
- 상품명
- 옵션명
- 옵션ID
- 현재SKU
- 추천SKU
- 상태
- 액션

### 상품별 그룹 테이블
그룹 기준:

```text
platform + platform_product_id
```

그룹 행 표시:
- 플랫폼
- 플랫폼 상품번호
- 상품명
- 옵션 수
- 매핑 수
- 미매핑 수
- 충돌 수
- 미매핑 선택 버튼

그룹 기능:
- 전체 선택
- 부분 선택
- 접기/펼치기
- 미매핑 옵션만 선택

## 3. SKU 검색 팝업

### 실행
각 옵션 행의 SKU 검색 버튼을 누르면 열린다.

### 이동
팝업 상단 `SKU 검색` 헤더를 드래그하여 이동할 수 있다.

### 검색 결과 표시
- 선택 순번
- SKU 코드
- 품목명
- 규격

### 단일 선택
체크된 옵션 행이 없으면 검색 결과 클릭 시 현재 행에 SKU를 입력한다.

### 다중 선택
체크된 옵션 행이 있으면 검색 결과 클릭 시 SKU 선택 목록에 추가한다.

동작:
1. 옵션 행 N개 선택
2. SKU 검색 결과 N개를 순서대로 선택
3. 선택 순서대로 적용
4. 옵션 행 순서와 SKU 선택 순서대로 1:1 입력

옵션 수와 SKU 선택 수가 다르면 적용하지 않는다.

## 4. 저장 버튼 차이

### 선택 일괄 매핑
체크된 옵션 행만 저장한다.

### 입력된 SKU 일괄저장
현재 화면에서 SKU 입력값이 있거나 변경된 행을 저장한다.

체크 여부와 무관하다.

- 비활성 구성

- 출고단가
- 구성품수
- 구성원가
- 마진
- 액션

### 상세 패널
- 구성품 SKU
- 구성품명
- 수량
- 원가
- 저장
- 삭제
