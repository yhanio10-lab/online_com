# 데이터모델

## products

플랫폼 상품 단위 데이터.

| 컬럼 | 의미 |
| --- | --- |
| id | 내부 상품 ID |
| platform | 플랫폼 |
| product_id | 플랫폼 상품번호 |
| seller_product_code | 판매자 상품코드 |
| product_name | 상품명 |
| status | 상품 상태 |
| category_code | 카테고리 코드 |
| sale_price | 판매가 |
| tax_type | 과세 정보 |
| brand | 브랜드 |
| manufacturer | 제조사 |
| created_at | 생성일 |
| updated_at | 수정일 |

유니크:

```text
platform + product_id
```

## product_options

플랫폼 상품 옵션 단위 데이터.

| 컬럼 | 의미 |
| --- | --- |
| id | 내부 옵션 ID |
| platform | 플랫폼 |
| product_id | products.id |
| option_id | 플랫폼 옵션 ID |
| option_name | 옵션명 |
| status | 옵션 상태 |
| option_price | 옵션가 |
| stock_quantity | 플랫폼 옵션 재고 |
| raw_option_name | 원본 옵션명 |
| created_at | 생성일 |
| updated_at | 수정일 |

유니크:

```text
platform + option_id
```

인덱스:

```text
product_options(platform, option_id)
```

## sku_master

이카운트 품목 마스터.

| 컬럼 | 의미 |
| --- | --- |
| sku_code | 이카운트 품번코드 |
| sku_name | 품목명 |
| spec | 규격 |
| unit | 단위 |
| barcode | 바코드 |
| purchase_price | 입고단가 |
| purchase_price_vat_included | 입고단가 VAT 포함 여부 |
| sale_price | 출고단가 |
| sale_price_vat_included | 출고단가 VAT 포함 여부 |
| item_type | 품목구분 |
| is_set | 세트 여부 |
| inventory_managed | 재고수량관리 여부 |
| location | 상품위치 |
| is_active | 활성 여부 |
| price_updated_at | 단가 수정일 |
| created_at | 생성일 |
| updated_at | 수정일 |

PK:

```text
sku_code
```

## sku_mappings

상품 옵션과 SKU의 매핑 정보.

| 컬럼 | 의미 |
| --- | --- |
| id | 매핑 ID |
| platform | 플랫폼 |
| product_option_id | product_options.id |
| sku_code | sku_master.sku_code |
| mapping_status | active/inactive/conflict |
| created_by | 생성자 |
| created_at | 생성일 |
| updated_at | 수정일 |

인덱스:

```text
sku_mappings(platform, product_option_id)
sku_mappings(sku_code)
```

## mapping_histories

SKU 매핑 변경 이력.

| 컬럼 | 의미 |
| --- | --- |
| id | 이력 ID |
| mapping_id | sku_mappings.id |
| old_sku_code | 이전 SKU |
| new_sku_code | 변경 SKU |
| changed_by | 변경자 |
| changed_at | 변경일 |
| reason | 사유 |

## sku_bundle_components

세트/번들 SKU 구성품.

| 컬럼 | 의미 |
| --- | --- |
| id | 구성 ID |
| parent_sku_code | 세트 SKU |
| component_sku_code | 구성품 SKU |
| component_quantity | 구성 수량 |
| is_active | 활성 여부 |
| created_at | 생성일 |
| updated_at | 수정일 |

유니크:

```text
parent_sku_code + component_sku_code
```
